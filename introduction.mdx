Use Tilebox to access datasets, connect your own data, build processing pipelines, launch jobs, and inspect results. You can work through the console, command line, SDKs, APIs, or coding agents. Each access point uses the same authentication, data access, and job history.

## Concepts

Tilebox consists of two primary modules:
