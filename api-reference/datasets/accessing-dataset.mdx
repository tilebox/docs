Once you have listed all available datasets, you can access a specific dataset by it's name from the datasets object.

<RequestExample>

```python Python (Sync)
dataset = datasets.open_data.asf.sentinel1_sar
# or any other dataset available to you
```

```python Python (Async)
dataset = datasets.open_data.asf.sentinel1_sar
# or any other dataset available to you
```

</RequestExample>
